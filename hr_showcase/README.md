# HR展示页使用说明

## 本地打开

双击 `index.html` 即可查看，不依赖Python或数据库。

## 发给HR的推荐组合

1. 将本目录部署为公开静态网站，获得一个可直接访问的HTTPS链接。
2. 录制90秒产品演示视频，上传到飞书、腾讯文档或网盘。
3. 简历中只放一个作品集链接，不直接发送代码压缩包。

## 90秒录屏顺序

- 0-15秒：解释日报分散、月底汇总困难。
- 15-35秒：在日报录入页展示结构化字段。
- 35-50秒：查看匿名客户历史和待跟进事项。
- 50-70秒：展示月报指标、图表和Excel/Word导出。
- 70-82秒：展示PubMed检索和来源保留。
- 82-90秒：说明使用虚构数据、68项测试通过及项目边界。

## 公开前检查

- 不上传项目根目录中的 `.env`。
- 不上传 `data/pharma.db`。
- 不上传真实客户、医生、患者或企业内部资料。
- 公开演示建议固定使用Mock模式。
